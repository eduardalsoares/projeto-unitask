# 📚 UniTask

Plataforma web para gerenciamento de tarefas acadêmicas, desenvolvida como
projeto da disciplina de Tecnologia para Front-End.

## 🎯 Objetivo

Centralizar e organizar atividades universitárias (leituras, exercícios,
apresentações e avaliações), ajudando estudantes a cumprirem prazos e
absorverem melhor o conteúdo.

## ✨ Funcionalidades

- ✅ Criar, editar e excluir tarefas
- ✅ Marcar tarefas como concluídas
- ✅ Definir prioridade e status das atividades
- ✅ Visualização em calendário mensal
- ✅ Dashboard com resumo de tarefas realizadas

## 🖥️ Telas

| Tela | Descrição |
|---|---|
| Login/Cadastro | Entrada do usuário |
| Inicial | Lista de atividades do dia |
| Tarefas | Lista organizada por dias |
| Calendário | Visualização mensal |
| Dashboard | Estatísticas de produtividade |
